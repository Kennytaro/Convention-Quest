## Convention-Quest
- Convention Quest is a game where you, as a convention go-er, have to go around different convention floors collecting different merch. Once you have collected all the merch that is on that floor, you move on to going to the next floor to collect even more. You need to maneuver around booths that have been set up and to avoid other convention goers who try to get in your way and prevent you from collecting all the merch. Collect all the merch around the convention to win, avoid the hazards that are in your way, run into them too many times, and you'll lose.

## Run
From this folder:
- `python -m pip install pygame`
- `python main.py`

## Versions
- pygame 2.6.1 (SDL 2.28.4, Python 3.12.3)

## OS
- Windows
- Mac

## Controls
- Arrow keys/WASD: move
- `Space`: start/restart
- `Esc`: pause/unpause
- `Q`: quit
- `P`: toggle particles
- `F`: toggle player flash
- `M`: toggle audio
