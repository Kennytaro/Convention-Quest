# Convention-Quest
- Collect all the merch around the convention to win, avoid the hazards that are in your way, run into them too many times, and you'll lose.

## Run
From this folder:
- `python -m pip install pygame`
- `python main.py`

## Controls
- Arrow keys/WASD: move
- `Space`: start/restart
- `Esc`: pause/unpause
- `Q`: quit

## Versions
- pygame 2.6.1 (SDL 2.28.4, Python 3.12.3)
